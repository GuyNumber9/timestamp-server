Timestamp Server

A simple express-based timestamp server.

Usages:

To return the current time: 

https://example.domain/api/timestamp
result: 
{
    unix	1572196758590,
    utc	"Sun, 27 Oct 2019 17:19:18 GMT"
}

To return the time at 123456789 miliseconds after the UNIX epoch: 

https://example.domain/api/timestamp/123456789
result: 
{
    unix:	123456789,
    utc:	"Fri, 02 Jan 1970 10:17:36 GMT"
}

To return the miliseconds after the UNIX epoch on 2010-11-21: 

https://example.domain/api/timestamp/2010-11-21
result: 
{
    unix:	1290297600000,
    utc:	"Sun, 21 Nov 2010 00:00:00 GMT"
}
